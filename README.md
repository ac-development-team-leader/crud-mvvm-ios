# Swift MVVM App 🚀

A simple iOS application built with **Swift** using the **MVVM Architecture** pattern.

---

## ✨ Features

- Clean MVVM Structure
- Easy to Understand
- Scalable Architecture
- Reusable Components
- Beginner Friendly

---

## 📂 Project Structure

```bash
Project/
├── Model/
├── View/
├── ViewModel/
├── Network/
├── Services/
├── Extensions/
└── Utilities/
```

---

## 🔄 MVVM Flow

```text
View ↔ ViewModel ↔ Model
```

- **View** → UI Layer
- **ViewModel** → Business Logic
- **Model** → Data Layer
